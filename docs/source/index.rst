i2b2 Implementation Guide for Japan
===================================

GitHub 👉 https://github.com/yuakagi/i2b2-jp

| i2b2は、米国で開発されたオープンソースの臨床データ管理システムです。 日本の医療機関での利用を促進するために、
i2b2の日本語化と、 日本の医療情報システムとの連携方法をまとめたドキュメントです。
| 東京大学医学部附属病院企画情報運営部のメンバーが中心となってこのプロジェクトを進めています。

| Docker containerを利用した実装例も提供しており、 簡単にi2b2サーバーを立ち上げて試すことができます。
